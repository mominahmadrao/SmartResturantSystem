# SmartResturantSystem

The Smart Restaurant System is a full-stack food ordering and delivery platform where customers place orders, admins manage menus, and riders handle deliveries - built with React, FastAPI, and PostgreSQL for a secure, responsive experience.
This Smart Restaurant System is created by 4 members.

The backend would be able to run after reinstalling virtual environment and deleting the existing one and installing all the required files into the venv throught the requirements.txt file. Activating the virtual environment after installation via

```powershell

/venv/Scripts/Activate

```

The db.sql is the exported file of the database that can be imported on other devices.

Then using the command below for FastAPI to run:

```powershell

uvicorn main:app --reload

```

For the frontend, navigate to the frontend/smart-restaurant folder and run:

```powershell

npm run dev

```
